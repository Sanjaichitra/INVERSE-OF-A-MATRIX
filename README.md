## Aim:
To write a python program to find the inverse of a matrix

### Step 2:

### Step 3:

### Step 4: 
End the program

## Program:
 # Program to find the inverse of a matrix.
 # Developed by:SANJAI S
 # Register Number:23013614
 ~~~
import numpy as np
A = np.array([[6,2,3],[3,1,1],[10,3,4]])
result = np.linalg.inv(A)
print(result)
~~~
